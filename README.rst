PyTETRIS
========

Yet another Tetris game written in Python.

Requirements
------------

* Python >= 3.7, < 3.8

Installing
----------

Use ``pip`` to download and install the game. ::

    $ pip install py-tetris

Running
-------

Just type ``pytetris`` to run the game. ::

    $ pytetris

Controls
--------

Use the arrow keys to control the pieces.
